# Setup

- instalar vivaldi
- logar no e-mail (password manager)
- logar no vivaldi, sincronizar
- abrir bookmarks de setup
- baixar [mr-nvimrc](link do github aqui)
- instalar nvim
- colocar mr-rc no ~/AppData/Local/nvim
